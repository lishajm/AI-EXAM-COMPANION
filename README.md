# AI Exam Companion

AI-powered learning assistant for competitive exam preparation.

## Features
- AI Mentor Chatbot
- Mock Test Generation
- Performance Analysis
- Login & Register System

## Tech Stack
- Python
- Flask
- HTML/CSS/JS
- Ollama / OpenAI API

## How to Run
1. Install requirements
2. Run: python app.py
3. Open http://127.0.0.1:5000
